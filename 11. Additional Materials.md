# Exception Handling (예외 처리)
```cpp
double average{};
try
{
    if (total == 0)
        throw 0; // throw exception
    average = sum / total; // not execute if total == 0
}
catch (int &ex)
{
    // cerr은 에러 로깅을 위한 출력
    cerr << "Can't divide by zero" << endl;
}
cout << "Program continues" << endl;
```
